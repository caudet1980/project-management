# Task Manager Full-Stack

A full-stack project management app with React frontend and .NET backend.

## Structure
- `src/` - React frontend
- `backend/` - .NET API
- `public/` - Static assets

## Setup

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd backend
dotnet restore
dotnet run
```

## Features
- Project management
- Task tracking
- User interface with Tailwind CSS
